# Skill: Diagnóstico de Sostenibilidad

## Propósito

Guías y mejores prácticas para realizar diagnósticos de iniciativas ESG y sostenibilidad empresarial.

## Cuándo Usar Este Skill

- Realizar diagnóstico de sostenibilidad
- Mapear iniciativas ESG
- Analizar brechas de evidencia
- Diseñar mapas de valor
- Identificar aliados clave

## Fase 1: Diagnóstico

### Mapeo de Iniciativas Vigentes

**Debes:**
- Mapear iniciativas sistemáticamente
- Identificar estado actual de cada iniciativa
- Documentar resultados y métricas existentes
- Identificar brechas de evidencia

**Preguntas clave:**
- ¿Cuáles son los compromisos ESG de la empresa?
- ¿Qué iniciativas existen y cuál es su estado?
- ¿Qué se mide y qué falta medir?
- ¿Qué iniciativas tienen resultados documentados?

### Mapeo de Aliados

**Debes:**
- Identificar aliados clave (internos y externos)
- Mapear roles y responsabilidades
- Analizar capacidad y disponibilidad
- Documentar relaciones existentes

**Preguntas clave:**
- ¿Quiénes son los aliados clave?
- ¿Qué roles juegan en las iniciativas?
- ¿Qué capacidad tienen?
- ¿Cómo se relacionan entre sí?

### Análisis de Conexión con Core del Negocio

**Debes:**
- Analizar cómo se conectan las iniciativas con el negocio
- Identificar casos de negocio
- Evaluar valor reputacional
- Mapear riesgos y oportunidades

**Preguntas clave:**
- ¿Cómo se conectan las iniciativas con el core del negocio?
- ¿Cuál es el valor de negocio de cada iniciativa?
- ¿Qué riesgos mitigan o qué oportunidades habilitan?
- ¿Cómo fortalecen la reputación?

### Creación de Mapa de Valor

**Debes:**
- Crear mapa de valor (problema-oportunidad-actores)
- Definir línea base IMPACTO
- Identificar prioridades operativas
- Documentar decisiones clave

**Entregables:**
- Mapa de valor
- Línea base IMPACTO
- Prioridades operativas

## Principios

- **Sistemático:** No dejes cabos sueltos
- **Práctico:** Cada análisis debe llevar a una decisión
- **Balanceado:** Integra negocio, impacto y datos
- **Evidencia honesta:** Medimos lo que importa
