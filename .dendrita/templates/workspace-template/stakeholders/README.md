# Stakeholders

This folder contains allies and stakeholder management documents.

## Purpose

Manage relationships, alliances, and stakeholder information:
- Allies mapping
- Contract templates
- Governance documentation
- Relationship tracking

## Structure

```
stakeholders/
├── allies-mapping.md
├── projects-governance.md
└── templates-contratos/
    └── ...
```

## Usage

1. Maintain `allies-mapping.md` with current allies
2. Document governance in `projects-governance.md`
3. Store contract templates in `templates-contratos/`

---

**This is a template folder. Customize according to your workspace needs.**
