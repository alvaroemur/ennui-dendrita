# Products

This folder contains the products portfolio and product documentation.

## Purpose

Store product documentation and portfolio information here. Each product should have its own folder with:
- `README.md` - Product description and documentation
- Related files and resources

## Structure

```
products/
├── [product-name-1]/
│   └── README.md
└── [product-name-2]/
    └── README.md
```

## Usage

1. Create a folder for each product
2. Document the product in `README.md`
3. Include product details, features, and usage

**Note:** Products are reusable offerings (services, tools, content, etc.) that can be delivered across multiple projects.
