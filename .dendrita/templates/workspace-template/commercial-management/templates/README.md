# Templates de Propuestas

Esta carpeta contiene templates reutilizables para la creación de propuestas comerciales.

## Templates Disponibles

### `proposal-base.md`
Template base genérico para propuestas comerciales. Incluye estructura estándar con secciones comunes.

### Templates Adicionales
Crear templates específicos según metodologías o tipos de propuestas del workspace (ej: Art Thinking, Design Thinking, etc.)

## Uso

1. **Copiar** el template apropiado a la carpeta de estado correspondiente en `proposals/`
2. **Renombrar** el archivo con el nombre del cliente y fecha/versión
3. **Completar** las secciones según la información del cliente
4. **Revisar** que todos los apartados estén presentes antes de enviar
