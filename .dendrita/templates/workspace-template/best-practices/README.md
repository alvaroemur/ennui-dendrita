# Best Practices

This folder contains templates and workspace-specific methodologies.

## Purpose

Store reusable templates and methodologies specific to your workspace here. Each practice should have its own folder with:
- `README.md` - Methodology documentation
- Templates and examples
- Related files

## Structure

```
best-practices/
├── [practice-type-1]/
│   ├── README.md
│   └── templates/
└── [practice-type-2]/
    └── ...
```

## Usage

1. Create a folder for each methodology type
2. Document the methodology in `README.md`
3. Include templates and examples
4. Reference from projects when applicable

## Examples

- `sustainability-diagnostic/` - Methodology for sustainability diagnostics
- `fundraising-proposal/` - Templates for fundraising proposals
- `mel-system/` - MEL system templates
