# Stakeholders

Esta carpeta contiene la gestión de aliados y stakeholders de la empresa.

## Estructura Esperada

- `allies-mapping.md` - Mapeo de aliados y relaciones
- `projects-governance.md` - Documentación de gobernanza (opcional)
- `templates-contratos/` - Templates de contratos y acuerdos (opcional)

## Cómo Usar

1. Mantén `allies-mapping.md` actualizado con nuevos aliados
2. Documenta acuerdos importantes en archivos específicos
3. Usa templates de contratos como base para nuevos acuerdos
