# Template Workspace

Este es un template de referencia para crear nuevos workspaces de empresas dentro de ennui-dendrita.

## Estructura

Cada workspace debe contener las siguientes carpetas:

- `active-projects/` - Proyectos en curso
- `archived-projects/` - Proyectos completados
- `best-practices/` - Templates y metodologías específicas de la empresa
- `company-management/` - Gestión general de la empresa (opcional)
- `stakeholders/` - Gestión de aliados y stakeholders
- `tools-templates/` - Herramientas y templates reutilizables
- `work-modes/` - Modos de trabajo especializados

## Archivos Opcionales

- `config-estilo.json` - Reglas de estilo específicas de la empresa

## Estructura de Proyectos

Cada proyecto dentro de `active-projects/` o `archived-projects/` debe seguir la estructura estándar:

- `master-plan.md` - Plan maestro del proyecto
- `current-context.md` - Estado actual y decisiones
- `tasks.md` - Lista de tareas con estado
- `README.md` - Documentación del proyecto (opcional)

## Cómo Usar Este Template

1. Copia toda la estructura de `workspaces/template/` a un nuevo workspace
2. Reemplaza "Template" con el nombre de la empresa
3. Personaliza los README.md de cada carpeta según las necesidades de la empresa
4. Agrega contenido específico en cada carpeta según corresponda
