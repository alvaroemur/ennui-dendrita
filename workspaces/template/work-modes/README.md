# Work Modes

Esta carpeta contiene los modos de trabajo especializados para la empresa.

## Propósito

Los modos de trabajo son "agentes especializados" que proporcionan contexto y metodología para tipos específicos de trabajo.

## Estructura Esperada

Cada modo debe ser un archivo `.md` que documente:
- El propósito del modo
- Cuándo activarlo
- Metodología y herramientas
- Prácticas recomendadas

## Modos Base

Para comenzar, puedes copiar los modos base de `workspaces/ennui/work-modes/` y adaptarlos a las necesidades de la empresa.
