# Tools & Templates

Esta carpeta contiene herramientas y templates reutilizables para la empresa.

## Contenido Esperado

Templates comunes incluyen:
- Reportes trimestrales
- Teoría de cambio
- Matrices de riesgos
- Checklists de kickoff
- Otros templates según necesidades

## Cómo Usar

1. Revisa los templates disponibles antes de crear nuevos documentos
2. Copia y personaliza según el proyecto específico
3. Actualiza templates cuando identifiques mejores prácticas
