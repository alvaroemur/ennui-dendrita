# Archived Projects

Esta carpeta contiene proyectos que han sido completados o archivados.

## Cómo Usar

- Mueve proyectos completados desde `active-projects/` a esta carpeta
- Mantén la estructura completa del proyecto (master-plan.md, current-context.md, tasks.md)
- Los proyectos archivados sirven como referencia histórica y aprendizaje
