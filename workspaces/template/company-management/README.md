# Company Management

Esta carpeta contiene documentos de gestión general de la empresa.

## Contenido Esperado

Documentos comunes incluyen:
- `pipeline.md` - Pipeline de proyectos y oportunidades
- `dashboard-proyectos.md` - Dashboard o resumen de proyectos
- Otros documentos de gestión según necesidades

## Cómo Usar

1. Mantén los documentos actualizados con el estado general de la empresa
2. Usa estos documentos como referencia para toma de decisiones
3. Actualiza regularmente el pipeline y dashboard
